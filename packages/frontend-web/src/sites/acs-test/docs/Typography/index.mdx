# Typography

## Design tokens

Typography is defined as Design Tokens and then installed as a dev dependency inside your project:

<pre>
  ```bash 
 
 npm install @bsbst/tokens <>
  ```
</pre>

### CSS variables are included in the core CSS

<pre>
```css
.bsbst-component {
    --_bold: var(--font-weight-bold); // set a private variable
    font-weight: var(--font-weight-bold); // directly use a root variable
}
```
</pre>

### Import tokens as ES6 Variables to use in JS

<pre>
```js
import {
  FONT_WEIGHT_SEMIBOLD,
} from '../../../data/tokens.es6';
```
</pre>
