# Colors

## Design tokens

Colors are defined as Design Tokens and then installed as a dev dependency inside your project:

<pre>
```bash
npm install @acs-test11/tokens
```
</pre>

### CSS variables are included in the core CSS

<pre>
```css
.acs-test-component {
    --_private: var(--color-gray-light24); // set a private variable
    color: var(--color-gray-dark32); // directly use a root variable
}
```
</pre>

### Import tokens as ES6 Variables to use in JS

<pre>
```js
import {
  COLOR_BRAND_BLUE,
} from '../../../data/tokens.es6';
```
</pre>
