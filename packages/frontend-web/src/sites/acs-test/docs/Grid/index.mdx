# Grid

## Grid Specs:

- CSS Grid
- Col Size Classes correspond to 1 through 12 columns
- Col Start Classes, there are 24 lines to choose from -- 24 lines allow more options for centering ODD col sizes
- Apply backcolors and edge to edge images to `acs-test-container`
- Max width of `acs-test-grid` is `1280px`

### Gap sizes:

- Default - 20px
- Medium - 12px
- Small - 5px

### Grid padding:

- Default - 50px
- Medium - 30px
- Small - 10px

### Column Usage:

- All grids must hae a `acs-test-container` surrounding the grid
- All columns should contain at least a default column class and can be further extended for medium (tablet) and small (mobile)

### Responsiveness:

- The grid uses container queries which means it is responsive to the container it is in. It is not responsive towards the viewport.

#### Classes

- Assign column width at all breakpoints: `acs-test-grid__col-{1-12}`
- Assign column width _upto_ `medium`: `acs-test-grid__col-sm-{1-12}`
- Assign column width _upto_ `large`: `acs-test-grid__col-md-{1-12}`

#### SASS/SCSS

- Assign column `@include make-col({1-12})`
- Assign column start `@include start-at-col({LINE_NUMBER})`
